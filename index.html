potato-load.js demo

Lazy load images without jquery

Note

This plugin will simply load images as soon as they are within the viewport. If you don't want layout thrashing set the <img>height and width attributes.

To run again, clear your cache and force refresh. The images should get fetched as you scroll around. Check activity in your favorite browser's network log. I like to use Chrome's inspector.
